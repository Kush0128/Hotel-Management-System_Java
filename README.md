# Hotel Management System

A desktop-based Hotel Management System developed using Java. This application helps hotel staff manage rooms, customers, employees, bookings, and other hotel operations through a user-friendly graphical interface.

## Features

- User Login System
- Add Customer Details
- Add Employee Details
- Add Driver Details
- Add Room Information
- View Customer Information
- View Employee Information
- View Room Information
- Search Available Rooms
- Update Room Status
- Update Check-In / Check-Out Details
- Reception Dashboard
- Hotel Management Dashboard

## Technologies Used

- Java
- Java Swing (GUI)
- JDBC
- MySQL Database
- Eclipse IDE
